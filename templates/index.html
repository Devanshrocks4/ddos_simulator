How to Use

  • Enter Target URL: Specify the URL you want to simulate the attack on.
  • Number of Requests: Input the number of requests you want to send in the attack.
  • Select Attack Type: Choose from SYN Flood, UDP Flood, or HTTP Flood.
  • Start Attack: Click on the desired attack type button to begin the simulation.
  • Stop Attack: Click on the "Stop Attack" button to halt the simulation.
  • View Results: The attack overview will show the total requests sent and a chart displaying the attack performance.
